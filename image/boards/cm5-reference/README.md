# CM5 reference board pack

Status: draft scaffold

This pack is for a fixed, named CM5 + carrier combination only.
Do not interpret it as generic CM5 support.

## Commercial rule

A CM-based commercial SKU should be tied to a single validated carrier, I/O layout, storage strategy, and network path.
